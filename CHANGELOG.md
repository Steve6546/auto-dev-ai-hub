# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-07-21

### Added

- Initial project structure and basic scaffolding.
- CI/CD with GitHub Actions.
- Linting and formatting with ESLint and Prettier.
- Docker and Docker Compose for local development.
- Initial infrastructure stubs for Terraform and Helm.
- Express server with /health and /metrics endpoints.
- Environment variable handling with dotenv.
- Logging with Winston.
- Unit tests with Jest.
- AI integration with OpenAI.
- Context management for conversations.
- Workflow for code, test, and review agents.
- Error handling and retry mechanisms.
- Integration tests for the agents.
- Secrets manager for API keys.
- Monitoring and alerting with Prometheus and Grafana.
- Centralized logging with Loki.
- Deployment automation with GitHub Actions.
- Versioning and releases with Semantic Release.
- Final documentation in TECH_STACK.md and docs/usage.md.
- E2E and smoke tests.

## [1.1.0] - 2025-07-22

### Added

- UI wireframing generation by AI agent.
- `CONTRIBUTING.md` and `ROADMAP.md` for project documentation.
- Centralized model configuration.

### Changed

- Updated dependencies to the latest versions.
- Improved error handling in agent scripts.
- Enhanced `README.md` with "Getting Started" and "Project Structure" sections.
- Added JSDoc comments to key files for better code clarity.
