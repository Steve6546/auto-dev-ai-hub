# AutoDev AI Hub

A brief description of your project.

## 🚀 Getting Started

To get started with AutoDev AI Hub, follow these steps:

1.  **Clone the repository:**
    ```sh
    git clone https://github.com/Steve6546/UIU123.git
    cd UIU123
    ```

2.  **Install dependencies:**
    ```sh
    npm install
    ```

3.  **Set up your environment:**
    -   Create a `.env` file in the root of the project.
    -   Add your `GEMINI_API_KEY` or `OPENAI_API_KEY` to the `.env` file.

4.  **Run the agent:**
    -   To run a specific round, use the `run-round` command:
        ```sh
        node scripts/ai-cli.js run-round 1
        ```

## 🏗️ Project Structure

The project is organized as follows:

-   **`src/`**: Contains the core source code.
    -   **`agents/`**: Holds the different AI agents (e.g., `CodeAgent`, `DocsAgent`).
    -   **`backend/`**: Contains the backend services, including the LLM service and secrets manager.
    -   **`frontend/`**: Contains the React/Electron frontend application.
-   **`scripts/`**: Contains helper scripts, including the main `ai-cli.js` and `run-agent.sh`.
-   **`project_tasks.yaml`**: Defines the project's backlog and tasks for the AI agent.
-   **`TECH_STACK.md`**: Documents the project's tech stack.
-   **`ui-wires/`**: Contains the UI wireframes generated by the AI agent.

## AI Integration

To use the AI features, you need to add your OpenAI API key to the `.env` file:

```

```
